PROJECT TITLE: Alpha Ecosystem Simulator
PURPOSE OF PROJECT: Ecosystems are a complex and delicate balancing game. The addition or removal of one species affects many other species with which it might compete for, or provide food. In this program you will get a chance to build your own ecosystem, and explore the effects of these interrelationships.
VERSION or DATE: v_3
HOW TO START THIS PROJECT: Open .jar file and follow instructions on screen
